# Sutanoya Chakraborty - Personal Homepage

This repository contains code to create the homepage.

**Link** - [sutanoya.github.io/website](https://sutanoya.github.io/website/)

## Instructions

* The main file with all contents is ```index.html```, text must be edited there
to edit the webpage.

> [!WARNING]
> To update the CV, replace the file ```assets/cv.pdf``` with the new CV, do not
change the file name or where it is placed.

* Commit changes and push. e.g.,
```
git add -A
git commit -m "Commit message here: what did you change"
git push
```

* Changes should be reflected in a few minutes.
